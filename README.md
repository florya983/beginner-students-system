Aplicación en Java para gestionar alumnos de un curso.

## Funcionalidades

- Carga de alumnos (nombre, nota, trabajo práctico)
- Modificacion nota
- Cálculo de promedio
- Búsqueda de nota máxima y mínima
- Identificación de alumnos aprobados
- Lista de todos los alumnnos

## Tecnologías

- Java(POO)
